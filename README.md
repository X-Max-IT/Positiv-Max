# Positivus Landing Page
Использование общедоступного лендинга для отработки продвинутой вёрстки с применением современных подоходов.

Технологии и навыки:
- **SCSS** - @use, @forward, _config.scss, миксины, переменные
- **БЭМ** - осмысленный нейминг классов, независимые блоки
- **Адаптив** - миксины для медиа-запросов, clamp(), Grid и Flexbox
- **Семантика и доступность** - details, dialog, visually-hidden
- **CSS-анимации** - бургер-меню, ховер-эффекты, плавные transitions
- Самостоятельная разработка блока Footer
